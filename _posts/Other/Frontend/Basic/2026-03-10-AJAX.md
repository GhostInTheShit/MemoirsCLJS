# AJAX

Asynchronous JavaScript And Xml 

不是一门语言是一个概念.

不用刷新整个页面.

`$.ajax({url:"",type:"",success:fun1,error:fun2})` 异步请求服务器

## 在Augular中使用AJAX

`npm install @angular/cli -g` 全局安装angular

`npm install typescript -g` 全局安装typescript 

`ng new app1`

`ng server` 编译和运行,开启内部开发服务器.4200端口

`ng g class Product` 创建 Product类

```ts
export class Product 
{
    _id:number;
    productname:string;
    cost:number;

    constructor(_id:number,productname:string,cost:number)
    {
        this._id=_id;
        this.productname=productname;
        this.cost=cost;
    }
}
```
