# JQuery

更简洁简短的js代码.

直接下载然后在`<head>`中引用

## 选择器--JQuery的特色

`$("#id")` id选择器,等同于`document.getElementById("id")` ,以下同理

`$("")` 标签选择器,可以多选,用逗号隔开.

`$(".class")` 类选择器

`$(JSObject)` 可以将js对象转换为JQuery对象

大多数函数可以链式调用.

### 子类选择器

`$(div p)` 子代选择器,div里面的p标签,**包括嵌套子代**

`$(div>p)` 直接子代选择器,不包括嵌套.

`$(#p3~p)` 同一个父元素,p3之后的p元素.

`$(#p3+p)` 同一个父元素,p3之后的第一个p元素.

### 属性选择器

`$("img[src='img1.jpg'],img[src='earth.gif']")` 属性选择器,可以多选

`$("img[src!='img1.jpg']` 反选

`$("img[src^='ear']` 以什么开始,这里以`ear`开始的src

`$("img[src$='jpg']` 以什么结束,这里以`ear`结束的后缀

`$("img[src*='ar']` 包含,这里包含`ar`的src

## 过滤选择器

`$("p:first")` 第一个p标签

`$("p:last")` 最后一个p标签 

`$("p:eq(4)")` equal,索引为4的p标签,从0开始

`$("p:odd")` 索引为奇数的p标签,一般用来设置多行表格背景色。

`$("p:even")` 索引为偶数的p标签

`$("p:gt(5)")` 索引大于5的p元素.不包括5

`$("p:lt(5)")` 索引小于5的p元素,不包括5

`$("p:not(p:eq(4))")` not 为反选,这里为索引不等于4的p标签

`$("p:contains("Services")")` 内容包含`Services`的P标签

`$("div:has('p')")` 包含P标签的div

`$("div:empty")` 空的div

`$("p:first-child")` 作为第一个子元素的p标签.

`$("p:last-child")` 作为最后一个子元素的p标签.

`$("#table td:nth-child(2)")` 这里为tr的第一个td,相对为父元素的第2个子元素.不是索引.

`$("p:only-child")` 作为唯一子元素的p标签.

## 效果

显示

*   `fadeIn(1000)` 1秒淡入,需要它默认是隐藏的

    `slideDown(1000)` 一秒下滑,也是淡入效果,滑出来

    `show(1000)` 一秒显示,逐渐变大的淡入 

隐藏

*   `fadeOut(1000)` 1秒淡出,需要它默认是隐藏的

    `slideUp(1000)` 一秒上划,也是淡入效果,滑出来

    `hide(1000)` 一秒隐藏,逐渐变大的淡入 

设置透明度

*   `fadeTo(1000,0.5)` 设置透明度和时间

开关

*   `toggle(1000)` 开关,显示和隐藏
    
    `fadeToggle(1000)` 淡入开关

    `slideToggle(1000)` 滑动开关

## JQueryUI

一个插件

`jquery-ui.js`

`jquery-ui.css`

### 导入

先导入jquery,再导入jquery-ui

```html
<script src="jquery-4.0.0.min.js"></script>
<script src="jquery-ui.js"></script>
<!--注意css不是js文件,所有不能使用script导入-->
<link rel="stylesheet" href="jquery-ui.css">
```

### 功能

`datepicker()` 日期选取器

`spinner()` 可以手动调节+1 -1的文本

`autocomplete()` autocomplete会使文本输入自动提示相关选项

`tooltip()` 将有title的元素,title放大

```html
<div id="div1" title="this is div 1">div 1</div>
<input type="text" id="txt1">
<input type="text" id="txt2">

<script>
    //在div1中放置日期选取器
    $("#div1").datepicker();

    //spinner为可以手动调节+1 -1的文本
    $("#txt1").spinner();

    //autocomplete会使文本输入自动提示相关选项.
    //这里用{}传是因为传的是一个对象,可以设置许多参数.
    var availableTags = ["Java", "JavaScript", "Python", "C#", "C++", "Ruby", "Go", "Swift"];
    $("#txt2").autocomplete({
        source: availableTags
    });

    //将有title的元素,title放大
    $(document).tooltip();

</script>
```

`resizable()` 可以手动调节大小

`droppable()` 可以被放置,而且触发函数,需要别的拖动进去

`draggable()` 可以被拖动

```html
<div id="div2">div2</div>
<div id="div3">div2</div>

<script>
    //可以手动调节大小
    $("#div2").resizable();
    //可以被放置,而且触发函数,需要别的拖动进去
    $("#div2").droppable({
        drop: function() {
            $(this).css("background-color", "red");
        }
    });
    //可以被拖动
    $("#div3").draggable();
</script>
```

`selectable()` 可以被选择,鼠标点的那一下是selecting,点完被选择的是selected

`sortable()` 可以被用户重新排列,不能和selectable一起用会冲突

```html
<head>
    <style type="text/css">
        #list1{
            background-color: #ccc;
            list-style: none;
            padding: 0%;
        }
        #list1 li{
            margin: 3px;
        }
        #list1 .ui-selecting{
            background-color: red;
        }
        #list1 .ui-selected{
            background-color: blue;
        }
        .class1{
            background-color: aqua;
        }
    </style>
</head>

<body>
<ol id="list1">
    <li class="class1">1</li>
    <li class="class1">2</li>
    <li class="class1">3</li>
    <li class="class1">4</li>
    <li class="class1">5</li>
</ol>

<script>
    //可以被选择,鼠标点的那一下是selecting,点完被选择的是selected
    //$("#list1").selectable();

    //可以被用户重新排列,不能和selectable一起用会冲突
    $("#list1").sortable();
</script>
```

`accordion()` 手风琴效果，使用`h1`或其他`h`,放在`div`前面

`tabs()` 标签页效果,使用`ul`,`li`放在`div`前面.

```html
<body>

div1
<div id="div1">
    <h1>section 1</h1>
    <div>section 1 content</div>
    <h1>section 2</h1>
    <div>section 2 content</div>
</div>

div2
<div id="div2">
    <ul>
        <li><a href="#tabl1">section 1</a></li>
        <li><a href="#tabl2">section 2</a></li>
    </ul>
    <div id="tabl1">section 1 content</div>
    <div id="tabl2">section 2 content</div>
</div>

<script>
$("#div1").accordion();
$("#div2").tabs();
</script>
```

`dialog` 对话框

将一个隐藏的`div`以对话框的方式显示出来.

`buttons` 参数,添加按钮和函数.

`$(this).dialog("close")` 注意这里的`this`指的是`div1`(也就是`$("#div1").dialog("close")`),是个特殊的设置.

```html
<body>

    <input type="button" id="btn1" value="按钮1">
    <span id="span1">Output</span>

    <div id="div1" title="Title Here" style="display: none">
        <p>这是一个对话框</p>
    </div>

<script>
    $("#btn1").click(function () {
        $("#div1").dialog({buttons:{"Ok": fun1, "Cancel": fun2}});
    });

    function fun1() {
        $("#span1").text("你点击了Ok按钮");
        $(this).dialog("close");
    }
    function fun2() {
        $("#span1").text("你点击了Cancel按钮");
        $(this).dialog("close");
    }
</script>
</body>
```
