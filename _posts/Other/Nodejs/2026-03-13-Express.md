# Express

底层调用http模块,类似`.net`的miniapi，内置`queryString`,`fs`等模块

## 开始

`var app=express()` 就是http的createServer,但是不用传func

`app.listen(port,func(){})` 监听

`app.get("/login",(req,res)=>{})` 当进入`/login`时候,query语句不影响(智能识别地址和查询).第二个是调用的方法.

`app.use((req,res)=>{})` 备用函数,~~如果route没有匹配则执行这个~~(其实是因为get和post里面终止了响应).放在最后可以被终止，放在前面会执行.

`app.use(express.json())` 使用中间件解析json

`app.use(express.urlencoded({extended: true}));` 使用中间件解析某中类型。

### Requset

`request.url` 请求url

`request.method` 请求方法

`request.headers` 请求头

`request.query` Get获得查询

`request.body` Post获得查询

### Response

`response.write()` 写入

`response.end()` 结束响应

`response.header()` 设置响应头

`response.status()` 设置状态码

`response.sendFile(__dirname+"path")` `__dirname`为当前文件夹(内置变量),发送文件,如html,也会自动结束响应.

`response.send()` write()+end()

## Router

```js
var adminRouter=express.router();
app.use("/admin",adminRouter);

//这里为/admin/dashboard
adminRouter.get("/dashboard",(req,res)=>{});
```

## EJS

`ejs` 模块

视图引擎,`html`后缀的页面改成`ejs`,支持html和js的混合,  
支持MVC,需要在根目录创建一个`views`文件夹

`<% code %>` 用这个嵌入js.

`<%=variable%>` 显示变量(多一个`=`)

```js
import express from 'express';
import ejs from "ejs";

var app =express();
//启用ejs视图渲染
app.set("view engine","ejs");

app.listen(7070,()=>{console.log("server is running at http://localhost:7070")});

app.get("/",(req,res)=>{
    var model={x:1,y:2};
    //渲染index.ejs,并且提供模型
    res.render("index",model);
});
```

嵌入局部视图

~~`<%include page.ejs%>` 已弃用~~

`<%-include("search")%>` 嵌入局部视图

## Mustache

`mustache-express`模块

视图引擎，同样需要views，和ejs同类型的框架.

视图后缀为`.mustache`

```js
{{#bool}}
html 
{{/bool}}
{{^bool}}
html
{{/bool}}

//等价于if-else

//会自动遍历
{{#array}}
html
{{/array}}

//嵌入
{{>file.mustache}}
```

### 示例
```js
import express from 'express';
//导入
import mustacheexpress from "mustache-express";

var app =express();
//注册一个引擎,ejs是亲儿子,所以不需要
app.engine("mustache",mustacheexpress());
//设置视图引擎为mustache
app.set("view engine","mustache");

app.listen(7070,()=>{console.log("server is running at http://localhost:7070")});

app.get("/",(req,res)=>{
    var model={x:1,y:2,z:[1,2,3],b:true};
    res.render("index",model);
});
```

```html
<!DOCTYPE html>
<html lang="en">
<head>

</head>
<body>
    {{x}}
    {{y}}

    {{#b}}
    true
    {{/b}}

    {{^b}}
    false
    {{/b}}

    {{z}}

    {{>search}}
</body>
</html>
```

## Handlebars

`express-handlebars` 模块

和`Mustache`非常相似，mustache的上位替代    

~~视图用`.hbs`后缀，局部视图用`.handlebars`后缀~~  
现在都用`.handlebars`后缀

需要在`views`文件夹中创建一个`partials`文件夹,用来放置局部视图

现在新版还需要在`views`中创建一个`Layouts`布局文件夹,里面至少要有一个`main.handlebars`布局视图
```js
//main.handlebars
//显示主体
{{{body}}}
```

### 示例
```js
import express from 'express';
//导入模块
import handlebars from "express-handlebars";

var app =express();

//注册引擎,使用模块里面的engine方法
app.engine("handlebars",handlebars.engine());
//设置视图引擎为刚刚创建的引擎
app.set("view engine","handlebars");

app.listen(7070,()=>{console.log("server is running at http://localhost:7070")});

app.get("/",(req,res)=>{
    var model={
    x:1,
    y:2,
    z:[1,2,3],
    b:true,
    student:{id:1,name:"tom",age:20}};
        
    res.render("index",model);
});
```

handlebars比mustache更加直观
```html
<body>
    {{x}}
    {{y}}

    <!--if语句-->
    {{#if b}}
    true
    {{else}}
    false
    {{/if}}

    {{#each z}}

    {{/each}}

    <!--插入局部视图-->
    {{>search}}

    <!--用于显示内部属性-->
    {{#with student}}
        {{id}}
        {{name}}
        {{age}}
    {{/with}}
</body>
```
