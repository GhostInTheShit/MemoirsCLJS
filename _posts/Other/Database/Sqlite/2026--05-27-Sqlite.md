# Sqlite

嵌入式数据库,**不需要后台服务**，单文件数据库.有Nuget包就行了.不需要去官网下载和安装。仅仅是使用一个进程就能访问.

如有类型,写的时候有，其实底层只有字符串和数字.

两种方案
*   `sqlite-net-pcl` 社区标准库,比EFCore更加轻量，有基础够用的LINQ功能.依赖注入也不要

*   `Microsoft.EntityFrameworkCore.Sqlite`  官方库，EFCore的Sqlite包  
    `Microsoft.EntityFrameworkCore.Design`  EFCore的设计迁移包
    `dotnet-ef`  命令行工具(不强制需要)

## sqlite-net-pcl方案

### Attribute 特性
```C#
//指定为主键,自增
[PrimaryKey,AutoIncrement]
public int Id { get; set;  } 
```

### 存放目录
```C#
//const是常量，编译时确定
public const string DatabaseName = "poetry.sqlite3";

//readonly是运行时确定的常量
//Path.Combine是用来跨平台拼接目录地址的.Windows和linux斜杠有正反
//Environment.SpecialFolder.LocalApplicationData 是一个枚举,指本地应用目录
//Environment.GetFolderPath 通过枚举得到具体目录
public static readonly string PoetryDbPath = Path.Combine(Environment.GetFolderPath(Environment.SpecialFolder.LocalApplicationData), DatabaseName);
```

### 创建连接
```C#
private SQLiteAsyncConnection _connection;

public SQLiteAsyncConnection Connection => _connection ??= new SQLiteAsyncConnection(PoetryDbPath);

public async Task InitializeAsync() => await Connection.CreateTableAsync<Poetry>();
```
