# NetWork

网络,内置非常简单易用的网络功能.许多功能都是已经写好的.

## 多人

首先需要同步的`GameObject`要打开网络设置,修改`NetWork Mode`

有三种网络模式
*   Never Network 没有网络
*   Network Object 网络对象,频繁的数据交换,一般给玩家角色
*   Network Snapshot 网络快照
   
如果一个`GameObject`为网络对象,那么它将自动同步信息.

## INetworkLisentener

可以用来写Network Manager,内置一些生命周期函数.

命名空间 `Sandbox.Component.INetworkListener` 

`OnActive(Connection channel)` 当有连接的时候触发

`Connection.DisplayName` 连接的用户名(默认为steam的)

`gameObject.NetworkSpawn(channel)` 为某个连接创建gameObject,并且设置为`Network Object`,不会将子类也设置.
