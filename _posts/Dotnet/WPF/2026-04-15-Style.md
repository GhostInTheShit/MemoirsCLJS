# Style

样式.

```xml
<StackPanel>
    <StackPanel.Resources>
        <!-- 在StackPanel范围内 -->
        <!-- 指定类型 -->
        <Style TargetType="TextBlock">
            <!-- 设置属性 -->
            <Setter Property="Foreground" Value="Gray" />
            <Setter Property="FontSize" Value="24" />
        </Style>
    </StackPanel.Resources>

</StackPanel>
```
也可以直接用Style修改.但是没啥意义,不如直接写在属性里面
```xml
<TextBlock Text="test">
    <TextBlock.Style>
        <Style>
            <Setter Property="TextBlock.FontSize" Value="36" />
        </Style>
    </TextBlock.Style>
</TextBlock>
```
也可以写在窗口级别,也可以写在App全局里面.

## 指定使用样式

指定哪些控件使用样式.

```xml
<Window.Resources>
        <Style x:Key="MyStyle" TargetType="TextBlock">
                <Setter Property="Foreground" Value="Red"></Setter>
        </Style>
</Window.Resources>
<DockPanel Margin="10">
        <TextBlock Style="{StaticResource MyStyle}">text</TextBlock>
</DockPanel>
```

## 样式触发器

属性触发
```xml
<TextBlock Text="text">
        <TextBlock.Style>
                <Style TargetType="TextBlock">
                        <Setter Property="Foreground" Value="Blue"></Setter>
                        
                        <Style.Triggers>
                                <Trigger Property="IsMouseOver" Value="True">
                                        <Setter Property="Foreground" Value="Red"></Setter>
                                </Trigger>
                        </Style.Triggers>
                </Style>
        </TextBlock.Style>
</TextBlock>
```

数据触发

```xml
<CheckBox Name="MyCheckBox" Content="Hello"></CheckBox>
<!-- 注意这里TextBlock不要直接写死text,因为它优先级比Trigger高 -->
<TextBlock>
        <TextBlock.Style>
                <Style TargetType="TextBlock">
                        <Setter Property="Foreground" Value="Blue"></Setter>
                        
                        <Style.Triggers>
                                <DataTrigger Binding="{Binding ElementName=MyCheckBox,Path=IsChecked}" Value="True">
                                        <Setter Property="Text" Value="yes"></Setter>
                                </DataTrigger>
                        </Style.Triggers>
                </Style>
        </TextBlock.Style>
</TextBlock>
```
