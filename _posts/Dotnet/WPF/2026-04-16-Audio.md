## MediaPlayer

可以使用`MP3`,`MPEG`,`M4a`等普遍音频

```csharp
private MediaPlayer MediaPlayer=new MediaPlayer();

private void ButtonBase_OnClick(object sender, RoutedEventArgs e)
{
    OpenFileDialog openFileDialog = new OpenFileDialog();
    openFileDialog.Filter = "mp3 files (*.m4a)|*.m4a";
    if (openFileDialog.ShowDialog() == true)
    {
        mediaPlayer.Open(new Uri(openFileDialog.FileName));
        //播放
        mediaPlayer.Play();
    }
}
```

### 精确控制

`mediaPlayer.Open(new Uri(openFileDialog.FileName))` 加载某个文件,是个异步方法,
`MediaPlayer.NaturalDuration.TimeSpan`在未加载完毕取不到.

`MediaPlayer.Play()` 播放
    
`MediaPlayer.Stop()`  停止播放

`MediaPlayer.Pause()` 暂停播放

`MediaPlayer.Position.ToString(@"mm\:ss")` 音乐播放到时间.

播放器加载完某个文件后执行的事件.
`MediaPlayer.MediaOpened += MediaOpenEventHanlder;` 
```C#
private void MediaOpenEventHanlder(object? sender, EventArgs e)
{
    //音乐的最大时间,必须加载完毕后才能取到
    MusicDuration = MediaPlayer.NaturalDuration.TimeSpan.ToString();
}
```
