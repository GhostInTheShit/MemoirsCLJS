## Socket服务器

IP其实不是设置的,是自己有的,端口的话,服务器是自己设置的,客户端是分配的.

1.  创建Socket 需要三个参数,IPV4协议，stream，TCP  
`Socket Server = new(AddressFamily.InterNetwork, SocketType.Stream, ProtocolType.Tcp)`

1.  设置服务端Socket  
    `Socket.bind(Endpoint)` 绑定监听ip和端口请求  

    `Endpoint` 抽象类,是网络端点  
    可以用`IPEndPoint`,它不是抽象类,由ip和port(端口)组成`IPEndPoint(IPAddress ip,Int port)`   

    `IPAddress` 由静态类得到 `IPAddress.Any`，`Any`指的是任意一个IP，因为你要接收任何IP的请求,端口自己自定义数字  

1.  开始监听  
    `Socket.Listen(Int num)` 打开监听,专门用来接收**连接**,之后就不能`Send`,`num` 最大监听(连接)数  

成为服务端的`Socket`相当于是个连接管理,不参与和具体客户端通信.

### 服务端方法

`Socket.accept()`
由SocketWactch接收,服务端接收并返回一个新的Socket作为通信Socket,每个连接都会创建一个socket，需要用list管理.

`Socket.acceptAsync()` 异步接收连接

### 客户端方法

`Socket.Connect(Endpoint)` 客户端new一个Socket,不需要bind和listen，使用Connect就能**成为**与服务器通信的Socket;

## Socket通信

`Socket.Send()`   
`Socket.SendAsync()` 异步  
发送消息，参数是`byte[]`  
可以用`Encoding.UTF8.GetBytes(string)` 把string转为byte[];

`Socket.Receive()`  
`Socket.ReceiveAsync()` 异步  
用来接收信息,需要一个byte[],他会把信息填充到byte[]中，但是这里的数组大小为你估值，因为不能提前知道别人发送的信息大小.  
`Encoding.UTF8.GetString(byte[])` 最后转为字符串读取

## Encoding.UTF8

`Encoding.UTF8.GetBytes(string)` 把string转成byte\[\]

`Encoding.UTF8.GetBytes(string,intIndex,length)`
转化string从第几个string开始，转化多少个string

`Encoding.UTF8.GetString(byte[])` 把byte\[\]转成string

`Encoding.UTF8.GetString(byte[],intIndex,length)`
转化byte\[\]从第几个byte开始，转化多少个byte

## 粘包

TCP底层优化,有可能会出现粘包,掉包，为了解决这种问题,在每次传递的`byte[]`开头插入几个字节来表明大小.

```C#
public class CharParse
{
    //使用list缓冲,易于添加或减少
    private List<byte> _buffer=new();

    //解析数据,开头4个字节表示信息字节长度，length是由socket读取得到的
    public List<string> ParseMessage(byte[] data, int length)
    {
        //先将数据加到缓存
        _buffer.AddRange(data[..length]);
        //准备最后返回的字符串列表
        List<string> messages=new List<string>();

        while (true)
        {
            //如果长度不够4个，那么连信息的长度都无法读出，直接退出
            if (_buffer.Count < 4) break;

            //长度有了，看看缓存长度够不够4+信息长度，如果不够说明拼不出一条完整信息。
            int dataLength = BitConverter.ToInt32(_buffer.ToArray(),0);
            if (_buffer.Count < 4+dataLength) break;

            //能够至少拼出一条信息的情况
            byte[] messageByte = _buffer.GetRange(4,dataLength).ToArray();
            //获得具体信息的是数据
            string message = Encoding.UTF8.GetString(messageByte);

            //添加到列表
            messages.Add(message);

            //将缓存中的数据去除，因为将于messages发出
            _buffer.RemoveRange(0, 4+dataLength);
        }
        return messages;
    }
}
```
发送信息的装包
```C#
public void SendMessage(string message)
{
    //将字符串转化为字节数组，用于传输
    byte[] data = Encoding.UTF8.GetBytes(message);

    //得到data的长度并且转化为字节数组用于拼接，这里data.Length是int，所以会是4个字节长度
    byte[] dataLength = BitConverter.GetBytes(data.Length);

    //新的字节数组.
    byte[] newData = new byte[4 + data.Length];

    //将长度的字节数组复制到新数组开头
    dataLength.CopyTo(newData, 0);
    //将信息字节数组复制到新数组后面
    data.CopyTo(newData, 4);

    //发送
    _socket.Send(newData);
}
```
