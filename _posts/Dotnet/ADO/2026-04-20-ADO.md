# ADO

ADO.NET数据访问技术

## 连接字符串

连接数据库只能连接已经存在的数据库,如果不存在先连接到`master`再创建.

### 本地字符串

只能本地访问  

`Data Source`,`Initial Catalog` `Integrated Security`

LocalDB 是VS自带的轻量SQLServer,便于开发测试。  
`@"Data Source=(localdb)\MSSQLLocalDB;Initial Catalog=MyDatabase;Integrated Security=True;"`

`@"Data Source=localhost;Initial Catalog=MyDatabase;Integrated Security=True;"`

### 通用字符串

任何时候都能用

`Server` `DataBase` `Uid` `Pwd`

`Server=.`代表本地服务器
`@"Server=.;Database=MyDatabase;Trusted_Connection=True;"`

## SqlCommand

第一个参数为sql语句,第二个参数为connect  
`SqlCommand cmd = new SqlCommand(sql, conn)`

### NonQuery方法

`ExecuteNonQuery()` 非查询语句,返回int行数,没有查询性能更好,可以用来`update`,`insert into`,`delete from`

```cs
public void Insert()
{
    var conn = new SqlConnection(ConnString);
    
    string sql = @"insert into student(student_number, name, gender, birth_date, phone, class_id) values(20180002,'bob','M','2000-10-20',13340091112,1);";

    
    //创建命令,并且将sql语句和connect传入
    SqlCommand cmd = new SqlCommand(sql, conn);
    conn.Open();
    //非查询语句,返回int行数
    var executeNonQuery = cmd.ExecuteNonQuery();
    conn.Close();
}
```

### Query

#### 单一查询

`ExecuteScalar()` 单一值 返回的是Object
```cs
public void GetSingleResult()
{
    SqlConnection conn = new SqlConnection(ConnString);
    string sql = "select count(*) from student";
    SqlCommand cmd = new SqlCommand(sql, conn);
    conn.Open(); 
    var executeReader = cmd.ExecuteScalar();
    conn.Close();
    Console.WriteLine(executeReader?.ToString());
}
```

#### 查询列表

`ExecuteReader()` 返回一个`SqlDataReader`类型游标.

`SqlDataReader`  
`Read()`前进到下一条数据  
`[index]`和`[columName]`都可以用来取值  
`FieldCount` 列的数量
`NextResult()` 转到下一个结果集,如果查多张表的话

```cs
public void GetReaderList()
{
    using SqlConnection conn = new SqlConnection(ConnString);
    string sql = @"select student.id,student_number,student.name,gender,birth_date,phone from student";
    using SqlCommand cmd = new SqlCommand(sql, conn);
    conn.Open();
    //返回只读数据集,也是一个必须被释放的资源
    using var reader = cmd.ExecuteReader();
    //循环读取,reader默认在-1行
    while (reader.Read())
    {
        //索引和列名都可以
        Console.WriteLine(reader[0]);
    }
}
```
如果返回有两张表,尽量一个表一个表查,**不建议同时查多张**.
```cs
public void GetMultiList()
{
    using SqlConnection conn = new SqlConnection(ConnString);
    string sql = "select * from student select * from class";
    using SqlCommand cmd = new SqlCommand(sql, conn);
    conn.Open();
    var sqlDataReader = cmd.ExecuteReader();
    while (sqlDataReader.Read())
    {
        for (int i = 0; i < sqlDataReader.FieldCount; i++)
        {
            Console.Write(sqlDataReader.GetName(i)+":");
            Console.Write(sqlDataReader[i]+",");
        }
        Console.WriteLine();
    }

    //转到下一张表
    if (sqlDataReader.NextResult())
    {
        while (sqlDataReader.Read())
        {
            for (int i = 0; i < sqlDataReader.FieldCount; i++)
            {
                Console.Write(sqlDataReader.GetName(i)+":");
                Console.Write(sqlDataReader[i]+",");
            }
            Console.WriteLine();
        }
    }
}
```

```cs
public void GetDataSet()
{
    using SqlConnection conn = new SqlConnection(ConnString);
    string sql = "select * from student ; select * from class";
    using SqlCommand cmd = new SqlCommand(sql, conn);
    conn.Open();

    //SqlDataAdapter适配器,传入command
    var adapter = new SqlDataAdapter(cmd);
    DataSet ds = new DataSet();
    //将数据填入DataSet,也可以传入第二个参数表名
    adapter.Fill(ds);

    //取出每张表
    foreach (DataTable table  in ds.Tables)
    {
        //每张表的每行数据
        foreach (DataRow dataRow in table.Rows)
        {
            //得到每一列,dataRow.Table是因为可能有多张表,要得到数据那张
            for (int i = 0; i < dataRow.Table.Columns.Count; i++)
            {
                Console.Write($"{dataRow.Table.Columns[i]}:{dataRow[i]}");
            }
            Console.WriteLine();
        }
    }
}
```
