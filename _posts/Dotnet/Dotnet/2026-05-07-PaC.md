# 爬虫

仅用于学习.

--------------------------
首先 有两种方法得到网页信息，  
1. 直接得到整个html然后剥离分析  
2. 得到接口，后端信息数据

相对来说获得接口更为直接一点，毕竟我们需要的就是信息数据.
可以F12仔细观察哪个请求会发送我们所需要的数据,在网络的Fetch/XHR里面.

找到那个接口后，我们需要用代码模拟浏览器发送请求那么就可以得到数据。

---------------------------
找到接口后最直接的是需要`HttpClient`作为网络请求,但是直接使用`HpptClient`的话不如依赖注入使用`HttpClientFactory`
```C#
ServiceCollection serviceCollection = new ServiceCollection();

//这里虽然方法名为AddHttpClient，但其实是用到了Factory，并且向里面添加
//这里分别创建了两个不同且独立的httplcient
serviceCollection.AddHttpClient("DSWZ", client =>
{
    client.Timeout = TimeSpan.FromSeconds(5);
});

serviceCollection.AddHttpClient("token", client =>
{
});
```
---------------------------
发送请求  

HttpRequestMessage和httpclient的区别  

HttpRequestMessage一般用来单次发送，设置不同的请求头或者Content
httpclient一般用来打地基，可以设置所有请求的基础地址或者一些共用设置

httpclient其实也可以直接发送请求(简单的，而且不能设置content).但是HttpRequestMessage可以更加细致一些.

```c#
//创建一个HttpRequestMessage，设置请求方法
HttpRequestMessage request = new HttpRequestMessage(HttpMethod.Get,"YourUrl");

//从工厂中得到httpclient,这里的工厂类用了依赖注册
_clientDSWZ = httpClientFactory.CreateClient("DSWZ");

//使用httpclient发送这个请求，并得到响应
HttpResponseMessage response = _clientDSWZ.Send(request);
    
if (response.IsSuccessStatusCode)
{
    return await response.Content.ReadAsStringAsync();
}
```
---------------------------
然后你会发现其中会有验证的问题(需要`authorization`或者其他)  

其实`authorization`和`Cookie`也会在某一个请求中发送，一般在你长时间未操作或者时间过长令牌无效后容易得到.

找到这个请求，使用Postman请求，多半会被拒绝，这个时候我们需要模拟浏览器行为，因为服务器一般会拒绝掉代码请求防止爬虫。

在F12中找到那个请求右键复制URL，到postman里面导入就能得到相同的请求头，使用这些和浏览器一样的请求头就能模拟成浏览器请求了，Postman一般会有几个默认请求头，要关闭掉，因为会直接告诉服务器我是postman不是浏览器。

再次发送，应该就能得到服务器发给你的token。然后我们要改成用代码发送

postam右面有一个箭头，打开 可以直接将刚才的请求头设置转化为C#的`HttpClient`设置，但是要注意`HttpClient`的设计十分严格，关于`body`的部分设置会报错，如果`content-type`。所以还需要修改.

将那些请求头和内容全部设置到`HttpRequestMessage`，发送就能得到`authorization`的json

---------------------------------------
要解析json，可以用系统自带的`System.Text.Json`

```C#
//将json转化为document
using var doc = JsonDocument.Parse(json);

//这里得到根节点
JsonElement rootElement = doc.RootElement;
//这里得到根节点里面的data节点，根据不同json有不同的层次，这里不一定相同
JsonElement dataElement = rootElement.GetProperty("data");
```

如果得到了一个里面是数组的节点，可以用EnumerateArray将他展开为数组,例如:
```C#
JsonElement.ArrayEnumerator arrayEnumerator = productsElement.EnumerateArray();

foreach (var item in arrayEnumerator)
{
}
```
