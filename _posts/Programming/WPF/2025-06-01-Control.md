Label 标签 **可以聚焦到相关的控件**

```xml
<!--名字前下划线,然后指定目标元素,alt+N就能聚焦到Tbox,N为_Name的首字母-->
<Label Content="_Name" Target="{Binding ElementName=Tbox}"></Label>
<TextBox x:Name="Tbox"></TextBox>

<!--如果Label里面有多个元素,要用到AccessText-->
<Label Target="{Binding ElementName=Tbox}">
    <StackPanel>
        <Image Source="some.png" />
        <AccessText Text="_Name:" />
    </StackPanel>
</Label>
<TextBox Name="Tbox" />
```

```C#
private void TextBox_SelectionChanged(object sender, RoutedEventArgs e)
{
    TextBox textBox = sender as TextBox;
    //txtStatus为另一个UI控件用来显示
    txtStatus.Text = "Selection starts at character #" + textBox.SelectionStart + Environment.NewLine;
    txtStatus.Text += "Selection is " + textBox.SelectionLength + " character(s) long" + Environment.NewLine;
    txtStatus.Text += "Selected text: '" + textBox.SelectedText + "'";
}

// SelectionStart，它给出了当前光标位置或是否有选择：它从什么位置开始。
// SelectionLength，它给出了当前选择的长度，如果有的话。 否则它将返回0。
// SelectedText，如果有选择，它会给我们当前选择的字符串。 否则返回一个空字符串。
```

动态加载,不是预先准备的，比如用户上传图片,要在C#代码中编写

```c#
private void ButtonBase_OnClick(object sender, RoutedEventArgs e)
{
    //创建一个文件对话框
    OpenFileDialog openFileDialog = new();
    //openFileDialog.ShowDialog()为显示窗体，并且接受你是否点击了确认。
    if (openFileDialog.ShowDialog() == true)
    {
        //openFileDialog.FileName为你点击的文件的完整文件名
        var fileName = openFileDialog.FileName;
        Console.WriteLine(fileName);//输出 C:\Users\YouKnowWho\Desktop\atlas.png
        
        Uri uri = new(fileName);
        Console.WriteLine(uri);//输出 file:///C:/Users/YouKnowWho/Desktop/atlas.png

        Image.Source = new BitmapImage(uri);


        //如果为程序内部的资源,直接传递Uri,要加上UriKind.Relative因为是相对路径
        //Uri resourceUri = new Uri("/Images/white_bengal_tiger.jpg", UriKind.Relative);
        //imgDynamic.Source = new BitmapImage(resourceUri);    
    }
}
```
`stretch` 拉伸

*   Uniform: 这是默认模式。 图片将自动缩放，将保留图片的宽高比。
*   UniformToFill: 图片将被缩放，以便完全填充图片区域。 将保留图片的宽高比。
*   Fill: 图片将缩放以适合图片控件的区域。 可能无法保留宽高比，因为图片的高度和宽度是独立缩放的。
*   None: 如果图片小于图片控件，则不执行任何操作。 如果它比图片控件大，则会裁剪图片以适合图片控件，这意味着只有部分图片可见。
