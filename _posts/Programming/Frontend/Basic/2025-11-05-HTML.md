# HTML

可以直接在浏览器中打开.

`<!DOCTYPE html>` 是一个声明.

`head` 用来描述网页细节,和导入`css`或者`js`.  
你只能使用`title`,`meta`,`script`,`style`,`link`这些标签.这是书写规范.

`body` 网页的内容展示,可以放其他的所有标签.

`meta` 元数据,可以为搜索引擎提供信息,可以指定`charset`字符集,  
提供各种可以搜索的关键词. 
```html
<!--关键字-->
<meta name="keywords" content="google,C#">
<!--网页描述-->
<meta name="description" content="Poplulation html">
<!--视口宽度为设备宽度-->
<meta name="viewport" content="width=device-width">
```
只能写在`head`.


## ASP中的运用 

**for**

```HTML
<!--当label被点击时,input会被聚焦-->
<label for="some">
<input id="some">
```

**form**

表单,只有有`name`的控件才能被提交

**select**

里面包含`option`

**textarea**

多行文本框

**input**

type 
*   radio 相同`name`的单选被绑定为一组
*   date
*   text
*   email
