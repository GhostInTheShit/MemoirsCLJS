## 内容管道

MGCB 会将Asset转化为xnb.

内置Asset

*   Effect（.fx）：一个着色器文件，通过控制GPU上的图形渲染方式来创建自定义视觉效果。
*   SpriteFont 描述（.spritefont）：定义文本在游戏中如何显示的配置文件，包括字符集和字体属性。
*   SpriteEffect（.fx）：专为2D精灵设计的着色器，用于创建特殊视觉效果。
*   XML Content（.xml）：用于存储游戏信息（如关卡、对话或配置设置）的结构化数据文件。
*   LocalizedSpriteFont Description（.spritefont）：用于创建支持多种语言字体的配置文件。

`Content.Load<T>("Path")` 加载

`Content.UnLoad()`卸载所有

## Draw

`spriteBatch.Begin()`

```C#
//按照layerDepth绘制,高的在上面,如果不写参数就是按照绘制顺序
SpriteBatch.Begin(sortMode: SpriteSortMode.FrontToBack);
```

`SpriteBatch.Draw(Texture, position, Color.White)` 

```C#
//缩放也与锚点有关
SpriteBatch.Draw(
    _logo,              // texture
    new Vector2(        // position
        Window.ClientBounds.Width,
        Window.ClientBounds.Height) * 0.5f,
    iconSourceRect,     // sourceRectangle
    Color.White,        // color
    0.0f,               // rotation
    new Vector2(        // origin 锚点
        iconSourceRect.Width,
        iconSourceRect.Height) * 0.5f,
    1.0f,               // scale
    SpriteEffects.None, // effects
    1.0f                // layerDepth
);
```

`SpriteBatch.End()`
